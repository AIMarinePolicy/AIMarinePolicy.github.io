About the Project

This project aims to advance AI-driven regulatory compliance in marine environmental policies, focusing on marine pollution and biodiversity regulation.

Project Objectives

  • Identify priority areas in marine pollution and biodiversity regulation via expert workshops.
  • Develop AI-driven compliance monitoring systems using Large Language Models.

Methodology

The project employs a structured two-phase approach:

  • Phase 1: Stakeholder workshops to refine regulatory focus.
  • Phase 2: AI-based compliance monitoring system development.

Timeline & Deliverables

Key milestones:

  • March 2025: Stakeholder Workshop
  • April 2025: Workshop Outreach Report & AI System Development
  • Dec 2025: Submission of Project Results for Publication
